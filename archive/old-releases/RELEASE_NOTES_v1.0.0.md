🎉 First official release of openclaw-video!

## Features
- 🎙️ OpenAI TTS for voice synthesis
- 📝 Whisper for timestamp generation
- 🎬 Remotion for video rendering
- 🤖 CLI tools and OpenClaw integration

## Installation

```bash
npm install -g openclaw-video
```

Or use directly with npx:

```bash
npx openclaw-video help
```

## Quick Start

1. Set your OpenAI API key:
```bash
export OPENAI_API_KEY=your_key_here
```

2. Generate a video from text:
```bash
openclaw-video generate "Your script text here"
```

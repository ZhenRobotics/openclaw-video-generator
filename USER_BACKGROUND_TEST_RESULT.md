- ✅ 背景和文字平衡

**如需调整**:

1. **如果背景太明显**（文字不够突出）
   ```bash

   --bg-overlay "rgba(0,0,0,0.7)"  # 加深遮罩
   ```

2. **如果想强调背景**（背景是重点）
   ```bash
   --bg-opacity 0.5-0.6  # 提高背景可见度
   --bg-overlay "rgba(10,10,15,0.4)"  # 浅色遮罩
   ```

3. **科技风格强化**（匹配科技主题）
   ```bash
   --bg-opacity 0.4
   --bg-overlay "rgba(0,20,40,0.6)"  # 蓝色调遮罩
   ```
